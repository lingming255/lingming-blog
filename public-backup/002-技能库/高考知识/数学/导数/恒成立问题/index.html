导数恒成立问题解法体系#

核心思路#

graph LR
    A[恒成立问题] --> B[分离参数法]
    A --> C[分类讨论法]
    B --> D[求函数最值]
    C --> E[找临界点]

一、分离参数法（首选方法）#

适用条件：参数能完全分离到不等式一侧
解题步骤：

  1. 将不等式变形为：$a \geq g(x)$ 或 $a \leq g(x)$
  2. 求 $g(x)$ 在定义域内的最值
  3. 根据不等式方向确定参数范围

高考经典模型：

  1. 单峰函数型：
    $f(x) \geq 0$ 恒成立 $\Rightarrow a \leq \min g(x)$
    示例：$x^2 - ax + 1 \geq 0$ → $a \leq x + \frac{1}{x}$ → $a \leq 2$

  2. 含参指数/对数型：
    $a \geq \frac{\ln x}{x}$ → 求 $g(x)=\frac{\ln x}{x}$ 最大值
    解法：$g’(x)=\frac{1-\ln x}{x^2}$ → 极大值点 $x=e$ → $a \geq \frac{1}{e}$

二、分类讨论法（分离困难时使用）#

适用条件：

  • 参数无法分离
  • 分离后函数过于复杂
  • 含多参数问题

解题步骤：

  1. 求导 $f’(x)$，确定讨论分界点（导数为0的点、定义域边界点）
  2. 按参数 $a$ 的取值分区间讨论函数单调性
  3. 验证每个区间是否满足恒成立条件

分类关键点：

函数特征 分界点选择 高考真题示例
二次项含参 判别式 $\Delta=0$ 2022全国乙卷·21
含 $e^x$/$lnx$ 导函数零点 $x_0$ 2021新高考Ⅰ卷·22
有绝对值 分段点 2020天津卷·20

五、实战示例（2023全国甲卷）#

题目：设 $f(x)=e^x - ax$，若 $f(x) \geq 1$ 对 $\forall x \geq 0$ 恒成立，求 $a$ 范围。

解法选择：参数可分离 → 分离参数法

  1. 分离：$a \leq \frac{e^x - 1}{x}$
  2. 构造：$g(x)=\frac{e^x - 1}{x}$
  3. 求导：$g’(x)=\frac{xe^x - (e^x-1)}{x^2} = \frac{e^x(x-1)+1}{x^2}$
  4. 最值：当 $x=1$ 时 $g’(x)=0$，$g(1)=e-1$
  5. 结果：$a \leq e-1$

命题陷阱：当 $x=0$ 时定义域间断，需单独验证（用极限得 $\lim_{x\to0}g(x)=1$）。

六、备考建议#

  1. 优先尝试分离参数（约占高考题70%）
  2. 分类讨论核心：抓住导函数零点与定义域关系
  3. 必刷真题：
    • 2022新高考Ⅰ卷·22（含参绝对值函数）
    • 2021全国甲卷·21（双参数问题）
    • 2019浙江卷·22（隐零点问题）

重点训练 “分离参数+最值分析” 和 “二次导函数分类讨论” 两类模型，覆盖90%高考题型。
