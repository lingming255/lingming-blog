光合作用 (Photosynthesis): 总反应式: CO₂ + H₂O –(光能, 叶绿体)–> (CH₂O) + O₂ 光反应阶段: 场所：叶绿体类囊体薄膜。 过程：水的光解 (H₂O → [H] + O₂)；ATP的合成。 能量转换：光能 → 活跃的化学能 (储存在ATP和[H]中)。 暗反应阶段: 场所：叶绿体基质。 过程：CO₂的固定 (CO₂ + C₅ → 2C₃)；C₃的还原 (C₃ –([H], ATP)–> (CH₂O) + C₅)。 能量转换：活跃的化学能 → 稳定的化学能 (储存在糖类中)。 细胞呼吸 (Cellular Respiration): 总反应式 (有氧): C₆H₁₂O₆ + 6O₂ + 6H₂O –(酶)–> 6CO₂ + 12H₂O + 大量能量 有氧呼吸三个阶段: 第一阶段：细胞质基质，葡萄糖 → 丙酮酸 + [H] + 少量能量。 第二阶段：线粒体基质，丙酮酸 + H₂O → CO₂ + [H] + 少量能量。 ...

对比项 原核细胞 (Prokaryote) 真核细胞 (Eukaryote) 本质区别 无核膜包被的细胞核 有核膜包被的细胞核 细胞器 仅有 核糖体 有核糖体、线粒体、叶绿体、内质网、高尔基体等复杂细胞器 遗传物质 拟核区有一个大型环状DNA，无染色体。 DNA与蛋白质结合形成 染色体，位于细胞核内。 细胞壁 主要成分为 肽聚糖 植物：纤维素和果胶；真菌：几丁质；动物：无。 代表生物 细菌、蓝藻、放线菌、支原体 (无细胞壁) 动物、植物、真菌 (酵母菌、霉菌、蘑菇) 共同点 都有细胞膜、细胞质、核糖体、以DNA作为遗传物质。 ### 易混淆生物辨析 酵母菌: 单细胞真菌，真核生物。 霉菌: 多细胞真菌 (少数单细胞)，真核生物。 蓝藻: 原核生物，因含叶绿素和藻蓝素能进行光合作用，但无叶绿体。 链霉菌: 放线菌的一种，原核生物，名字带“霉”但属于细菌。

* 双层膜: * 线粒体 (Mitochondrion): 有氧呼吸的主要场所，细胞的“动力车间”。含少量DNA和核糖体。 * 叶绿体 (Chloroplast): 光合作用的场所。植物绿色部分的细胞含有。含少量DNA和核糖体。 单层膜: 内质网 (Endoplasmic Reticulum): 脂质合成的“车间”，蛋白质的加工和运输通道。 高尔基体 (Golgi Apparatus): 对来自内质网的蛋白质进行再加工、分类和包装。与植物细胞壁的形成有关。 溶酶体 (Lysosome): 内部含有多种水解酶，细胞的“消化车间”。 液泡 (Vacuole): (植物细胞) 调节细胞内环境、维持细胞形态。 无膜结构: 核糖体 (Ribosome): 蛋白质的“合成机器”。 中心体 (Centrosome): (动物和低等植物细胞) 与细胞有丝分裂有关。 细胞核 (The Nucleus) 功能: 遗传信息库，细胞代谢和遗传的控制中心。 结构: 核膜 (Nuclear Envelope): 双层膜，将核内物质与细胞质分开。 核孔 (Nuclear Pore): 核膜上的孔道，实现核质间频繁的物质交换和信息交流 (大分子通道)。 染色质 (Chromatin): 主要由 DNA 和 蛋白质 组成。是遗传信息的载体。易被碱性染料染成深色。 染色质 vs 染色体：同一物质在细胞不同时期的两种形态。 核仁 (Nucleolus): 与某种RNA的合成以及核糖体的形成有关。

考点 创立者与过程: 1665年，罗伯特·胡克 (Hooke) - 首次发现并命名“细胞(Cell)”。 安东尼·列文虎克 (Leeuwenhoek) - 首次观察到活细胞。 19世纪30年代，施莱登 (Schleiden) & 施旺 (Schwann) - 共同提出细胞学说核心内容。 魏尔肖 (Virchow) - 修正并补充：“细胞通过分裂产生新细胞”。 核心内容: 一切动植物都由细胞发育而来，并由细胞及细胞产物所构成。 细胞是一个相对独立的单位，既有它自己的生命，又对与其他细胞共同构成的整体的生命起作用。 新细胞可以从老细胞中产生。 科学价值与意义: 揭示了生物体结构的统一性。 使生物学研究进入到细胞水平。 为后续的进化论、遗传学等奠定了基础。

细胞增殖 (有丝分裂): 周期: 分裂间期 (主要进行DNA复制和相关蛋白质合成) + 分裂期。 分裂期各阶段特征 (重中之重): 前期: 膜仁消失，两体出现 (染色体、纺锤体)。 中期: 形态稳定，数目清晰，赤道板齐。 (观察染色体最佳时期) 后期: 着丝点分，数目加倍，移向两极。 末期: 膜仁再现，两体消失。 动植物细胞有丝分裂的区别: 前期纺锤体形成方式不同；末期细胞质分裂方式不同。 细胞分化 (Cell Differentiation): 实质: 基因的选择性表达。 特点: 稳定性、持久性、不可逆性 (通常)。 结果: 形成不同形态、结构和功能的细胞、组织和器官。 细胞衰老与凋亡: 衰老特征: 水分减少、酶活性降低、色素积累、细胞核固缩、膜通透性改变。 细胞凋亡 (Apoptosis): 由基因决定的细胞自动程序性死亡过程，是正常的生命现象 (如蝌蚪尾巴消失)。它与细胞坏死有本质区别。 细胞癌变: 根本原因: 原癌基因和抑癌基因发生突变。 特征: 无限增殖、形态结构显著变化、细胞表面糖蛋白减少 (导致易扩散转移)。

分泌蛋白的合成与运输 路径: 核糖体 (合成肽链) → 内质网 (加工、折叠) → 高尔基体 (再加工、分类、包装) → 细胞膜 (通过胞吐分泌出去)。 能量供应: 全程由 线粒体 提供能量 (ATP)。 穿膜层数: 0层。整个过程通过囊泡运输，体现了生物膜的流动性。 ## 11. 物质的跨膜运输 分类依据: 是否需要载体、是否消耗能量、运输方向。 物质运输 ├── 大分子: 胞吞/胞吐 (依赖膜流动性, 耗能) └── 小分子 ├── 被动运输 (顺浓度梯度, 不耗能) │ ├── 自由扩散 (不需载体, 如O₂, CO₂, 苯) │ └── 协助扩散 (需载体/通道, 如葡萄糖进入红细胞) └── 主动运输 (逆浓度梯度, 需载体, 耗能, 如Na⁺-K⁺泵) 渗透作用 (Osmosis): 特殊的扩散，指水分子或其他溶剂分子通过半透膜的扩散。 条件：半透膜 + 膜两侧存在浓度差。 酶与ATP (补充) 酶 (Enzyme): 本质: 绝大多数是蛋白质，少数是RNA。 作用: 催化剂，显著降低化学反应的活化能。 特性: 高效性、专一性、作用条件温和 (受温度、pH影响)。 对比项 酶 (Enzyme) 无机催化剂 (Inorganic Catalyst) 化学本质 绝大多数是蛋白质，少数是RNA 多为金属离子或简单的无机物 催化效率 高效性 (催化效率极高) 催化效率相对较低 专一性 高度专一 (一种酶通常只催化一种或一类反应) 通常无专一性或专一性很低 作用条件 温和 (常温、常压、适宜的pH) 通常需要高温、高压等剧烈条件 作用机理 显著降低化学反应的活化能 也能降低活化能，但程度远不如酶 共同点 催化前后自身性质和数量不变，都能降低反应活化能，提高反应速率。 ATP (三磷酸腺苷): 结构简式: A-PPP (A:腺苷, P:磷酸基团, ~:高能磷酸键)。 功能: 生命活动的 直接能源物质。 ATP与ADP的循环: ATP → ADP + Pi + 能量 (用于生命活动) ADP + Pi + 能量 → ATP (能量来自光能或化学能) 这是一个循环利用的过程，保证了能量的持续供应。

模型: 流动镶嵌模型 (Singer & Nicolson)。 组成: 基本骨架: 磷脂双分子层。 功能承担者: 蛋白质 (镶嵌、贯穿或覆盖)。 外表面: 糖蛋白 (由蛋白质和糖类结合而成，也叫糖被)，进行细胞识别、保护和润滑。 结构特点: 具有一定的流动性 (是细胞融合、胞吞胞吐的基础)。 功能特点: 选择透过性 (是活细胞的标志)。 功能: 将细胞与外界环境分隔开。 控制物质进出细胞。 进行细胞间的信息交流。
