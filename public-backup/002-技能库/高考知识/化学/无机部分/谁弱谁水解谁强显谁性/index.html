第一部分：“谁弱谁水解”#

这句话的意思是：当一种盐溶解在水中时，构成这种盐的离子中，哪一部分来自于“弱”酸或“弱”碱，哪一部分就会和水发生反应（即水解）。

我们先要有一个共识：“强”的东西在水里是完全电离的，非常稳定，懒得和水反应。而“弱”的东西总想变回稳定的分子状态，所以会去“抢”水里的 H⁺ 或 OH⁻。

我们来看四种类型的盐：#

  1. 强酸强碱盐 (如 NaCl, KNO₃, BaSO₄)

    • Na⁺ 来自强碱 NaOH，很稳定。
    • Cl⁻ 来自强酸 HCl，很稳定。
    • 两个都是“强者”，都很稳定，谁都不水解。
    • 结果：溶液呈中性。

  2. 强酸弱碱盐 (如 NH₄Cl, CuSO₄, AlCl₃)

    • Cl⁻ 来自强酸 HCl，很稳定，不水解。
    • NH₄⁺ 来自弱碱 NH₃·H₂O (氨水)。它是“弱者”，不稳定，它总想变回氨水分子。
    • 怎么变回去？ 它会去抢水电离出的 OH⁻： NH₄⁺ + H₂O ⇌ NH₃·H₂O + H⁺
    • 你看，它为了自己变稳定，把水里的 OH⁻ 抢走了，导致溶液里剩下了多余的 H⁺。
    • 结论：“弱碱”对应的阳离子发生了水解，使溶液显酸性。

  3. 强碱弱酸盐 (如 CH₃COONa, Na₂CO₃, NaAlO₂)

    • Na⁺ 来自强碱 NaOH，很稳定，不水解。
    • CH₃COO⁻ (醋酸根) 来自弱酸 CH₃COOH (醋酸)。它是“弱者”，不稳定，总想变回醋酸分子。
    • 怎么变回去？ 它会去抢水电离出的 H⁺： CH₃COO⁻ + H₂O ⇌ CH₃COOH + OH⁻
    • 你看，它为了自己变稳定，把水里的 H⁺ 抢走了，导致溶液里剩下了多余的 OH⁻。
    • 结论：“弱酸”对应的阴离子发生了水解，使溶液显碱性。

  4. 弱酸弱碱盐 (如 CH₃COONH₄)

    • NH₄⁺ 来自弱碱，想抢 OH⁻。
    • CH₃COO⁻ 来自弱酸，想抢 H⁺。
    • 两个都是“弱者”，都要水解。
    • 溶液的酸碱性取决于它俩谁抢得更厉害（即酸和碱的相对强弱），高中阶段一般不深入讨论，默认为中性或不做要求。

总结第一部分：“谁弱谁水解” 指的是盐离子中，来源于弱酸的阴离子 或 来源于弱碱的阳离子 会与水发生反应。

第二部分：“谁强显谁性”#

这句话是“谁弱谁水解”的结果推论，用来快速判断溶液最终的酸碱性。

  • 强酸弱碱盐: “强酸”赢了，所以溶液显酸性。 (如 NH₄Cl，HCl是强酸，所以显酸性)
  • 强碱弱酸盐: “强碱”赢了，所以溶液显碱性。 (如 NaAlO₂，NaOH是强碱，所以显碱性)
  • 强酸强碱盐: 两边都是“强者”，势均力敌，谁也赢不了，所以溶液显中性。 (如 NaCl)
  • 弱酸弱碱盐: 两边都是“弱者”，打架结果不确定，看它俩谁相对更强一点，所以酸碱性不确定。

举例巩固#

  • NaAlO₂ (偏铝酸钠)

    • 拆分: Na⁺ (来自强碱NaOH) 和 AlO₂⁻ (来自弱酸HAlO₂/Al(OH)₃)。
    • 判断水解: Na⁺强，不水解。AlO₂⁻弱，水解。 -> “谁弱谁水解”
    • 判断酸碱性: 强碱NaOH vs 弱酸Al(OH)₃，“强碱”赢了。-> “谁强显谁性”
    • 结论: 溶液显碱性。
    • 水解方程式: AlO₂⁻ + 2H₂O ⇌ Al(OH)₃ + OH⁻

  • FeCl₃ (氯化铁)

    • 拆分: Fe³⁺ (来自弱碱Fe(OH)₃) 和 Cl⁻ (来自强酸HCl)。
    • 判断水解: Fe³⁺弱，水解。Cl⁻强，不水解。
    • 判断酸碱性: 弱碱Fe(OH)₃ vs 强酸HCl，“强酸”赢了。
    • 结论: 溶液显酸性。
    • 水解方程式: Fe³⁺ + 3H₂O ⇌ Fe(OH)₃ + 3H⁺
